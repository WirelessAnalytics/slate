
# Authentication

- HTTP Authentication, scheme: basic 

- HTTP Authentication, scheme: bearer 

* API Key (ApiKeyQuery)
    - Parameter Name: **api_key**, in: query. 

* API Key (ApiKeyHeader)
    - Parameter Name: **X-DreamFactory-API-Key**, in: header. 

* API Key (SessionTokenQuery)
    - Parameter Name: **session_token**, in: query. 

* API Key (SessionTokenHeader)
    - Parameter Name: **X-DreamFactory-Session-Token**, in: header.
